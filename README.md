## Rodando o projeto

### Usando o node

Primeiro, instale as dependências:

```sh
npm install
```

ou

```sh
yarn
```

Depois, rode com o npm o script gulp ou o start:

```sh
npm run gulp
# ou
npm start
```

Com yarn:

```sh
yarn gulp
# ou
yarn start
```

### Usando Docker

Simplesmente rode:

```sh
docker-compose up
```
